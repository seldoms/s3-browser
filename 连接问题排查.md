# 🔧 连接问题排查指南

## ❌ 错误: "Invalid URL"

### 原因
Endpoint 地址格式不正确

### 解决方法

#### ✅ 正确格式
```
https://obs.cn-north-4.myhuaweicloud.com
```

#### ❌ 错误格式
```
obs.cn-north-4.myhuaweicloud.com          ❌ 缺少 https://
https://obs.cn-north-4.myhuaweicloud.com/ ❌ 末尾有斜杠
http://obs.cn-north-4.myhuaweicloud.com   ⚠️  应该用 https
```

### 快速修复
系统现在会自动帮您:
- 自动添加 `https://` (如果您忘记写)
- 自动移除末尾的 `/`

**建议**: 直接复制粘贴华为云控制台提供的完整地址

---

## ❌ 错误: "Access Denied" 或 "403"

### 原因
Access Key 或 Secret Key 不正确,或者权限不足

### 解决方法
1. 检查 AK/SK 是否复制完整
2. 确认密钥没有过期
3. 在华为云控制台检查权限设置

---

## ❌ 错误: "Bucket not found" 或 "404"

### 原因
Region 填写错误

### 解决方法
1. **让系统自动识别**: 输入正确的 Endpoint,Region 会自动填充
2. **手动检查**: 登录华为云控制台,查看存储桶所在区域

**常见错误**:
- Endpoint 是北京 (`cn-north-4`),但 Region 填成了上海 (`cn-east-3`)

---

## ❌ 错误: "Network Error" 或 "连接超时"

### 原因
网络问题或 Endpoint 地址错误

### 解决方法
1. 检查网络连接
2. 确认 Endpoint 地址可以访问
3. 如果在公司内网,检查防火墙设置

---

## 🔍 如何获取正确的 Endpoint?

### 方法一: 华为云控制台
1. 登录华为云控制台
2. 进入"对象存储服务 OBS"
3. 点击任意存储桶
4. 在"基本信息"中查看"Endpoint"

### 方法二: 华为云文档
访问: https://developer.huaweicloud.com/endpoint?OBS

---

## 💡 测试连接的步骤

1. **先填写 Endpoint**
   - 系统会自动识别 Region
   
2. **检查 Region 是否正确**
   - 应该自动填充,无需手动修改
   
3. **填写 Access Key 和 Secret Key**
   - 从华为云控制台复制粘贴
   
4. **点击"添加账户"**
   - 如果成功,会跳转到文件管理器
   - 如果失败,会显示具体错误信息

5. **查看错误提示**
   - 页面顶部会显示红色错误框
   - 根据错误信息调整配置

## 🆘 还是连接不上?

请检查以下信息:

- [ ] Endpoint 地址是否以 `https://` 开头?
- [ ] Endpoint 地址是否包含正确的区域代码?
- [ ] Access Key 是否完整复制(没有多余空格)?
- [ ] Secret Key 是否完整复制(没有多余空格)?
- [ ] 密钥是否已激活(未过期)?
- [ ] 网络是否正常?

如果以上都确认无误,可以:
1. 尝试在华为云控制台直接访问存储桶
2. 检查华为云账户状态
3. 联系华为云技术支持

---

**提示**: 使用"重置"按钮可以清空所有字段,重新开始配置。
