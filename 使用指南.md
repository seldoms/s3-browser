# S3 Browser - 快速使用指南

## 📋 字段说明

### 1. **显示名称**
- 用途:为这个连接起一个容易识别的名字
- 示例:`我的华为云存储`、`公司备份账户`

### 2. **账户类型**
- 用途:选择您使用的云服务商
- 选项:
  - Amazon S3 Storage (亚马逊)
  - 华为云 OBS
  - MinIO (私有部署)
  - 其他 S3 兼容存储

### 3. **REST Endpoint** (服务地址)
- 用途:S3 服务的访问地址
- 常见示例:
  - 华为云北京: `https://obs.cn-north-4.myhuaweicloud.com`
  - 华为云上海: `https://obs.cn-east-3.myhuaweicloud.com`
  - AWS 美东: `https://s3.amazonaws.com`
  - MinIO: `http://your-server:9000`

### 4. **Region** (区域) ⭐
- **作用**: 指定数据中心的地理位置
- **为什么需要**: 
  - S3 服务在全球有多个数据中心
  - 不同区域的数据是隔离的
  - 必须指定正确的区域才能访问对应的存储桶
  
- **常见区域代码**:
  
  **华为云 OBS**:
  - `cn-north-1` - 北京一
  - `cn-north-4` - 北京四
  - `cn-east-2` - 上海二
  - `cn-east-3` - 上海一
  - `cn-south-1` - 广州
  
  **AWS S3**:
  - `us-east-1` - 美国东部(弗吉尼亚)
  - `us-west-2` - 美国西部(俄勒冈)
  - `ap-northeast-1` - 亚太(东京)
  - `ap-southeast-1` - 亚太(新加坡)
  
  **MinIO**:
  - 通常使用 `us-east-1` (默认值)

### 5. **Access Key ID** (访问密钥 ID)
- 用途:身份验证的用户名
- 格式:一串字母和数字组合
- 示例:`AKIAIOSFODNN7EXAMPLE`

### 6. **Secret Access Key** (私有访问密钥)
- 用途:身份验证的密码
- 格式:一串较长的字母、数字和符号
- ⚠️ **重要**:这是敏感信息,请妥善保管

## 🚀 快速开始

### 华为云 OBS 配置示例

1. 登录华为云控制台
2. 进入"对象存储服务 OBS"
3. 点击"访问密钥" → "新增访问密钥"
4. 记录下 AK 和 SK

**填写信息**:
```
显示名称: 华为云北京
账户类型: 华为云 OBS
REST Endpoint: https://obs.cn-north-4.myhuaweicloud.com
Region: cn-north-4
Access Key ID: 您的 AK
Secret Access Key: 您的 SK
```

### AWS S3 配置示例

**填写信息**:
```
显示名称: AWS 美东
账户类型: Amazon S3 Storage
REST Endpoint: https://s3.amazonaws.com
Region: us-east-1
Access Key ID: 您的 AWS Access Key
Secret Access Key: 您的 AWS Secret Key
```

## ❓ 常见问题

**Q: Region 填错了会怎样?**  
A: 会导致连接失败,提示"存储桶不存在"或"访问被拒绝"

**Q: 如何查看我的 Region?**  
A: 在云服务商控制台查看存储桶详情,通常会显示所在区域

**Q: Endpoint 和 Region 有什么关系?**  
A: Endpoint 包含了 Region 信息。例如 `obs.cn-north-4.myhuaweicloud.com` 中的 `cn-north-4` 就是 Region

**Q: 连接失败怎么办?**  
A: 
1. 检查 Endpoint 地址是否正确
2. 确认 Region 与 Endpoint 匹配
3. 验证 Access Key 和 Secret Key 是否有效
4. 检查网络连接

## 🎯 使用技巧

- **首次使用**: 建议先用默认的华为云配置测试
- **自动记住**: 登录成功后,凭证会自动保存,下次打开无需重新输入
- **清除凭证**: 点击"重置"按钮可以清除保存的凭证
- **多账户**: 可以保存多个不同云服务商的配置
- **安全性**: 凭证仅保存在本地浏览器,不会上传到服务器
